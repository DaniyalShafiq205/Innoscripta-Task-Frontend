# Innoscripta Frontend App

## React App Setup

Min Requirements
 
`Docker and Docker-Compose`

## Installation

### Install my-project with composer.phar

`cd Innoscripta-Task-Frontend`

`docker-compose up  -d`


### Down Containers
`docker-compose down`

## After it your app will serve at: 
http://localhost:3030/ if you go with defaults settings
